How Alerts Work
The backend checks configured alert rules every 5 minutes against today's log data pulled from S3. When a threshold is breached, an HTTP POST is sent to your webhook URL with a Slack-compatible JSON payload. You can use Slack Incoming Webhooks, Microsoft Teams Connectors, or any custom endpoint.

Supported Metrics:
errors — total log lines with level=ERROR or Razorpay error_code set
error_rate — errors as percentage of total requests
payments_failed — count of Razorpay responses with non-null error_code
total_requests — total log entries in the file
warnings — total log lines with level=WARN
